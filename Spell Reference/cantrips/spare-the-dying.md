# Spare the Dying

- [ ] Homebrew

***0th-level necromancy***

**Casting Time:** action

**Duration:** instantaneous

**Range:** touch

**Tags:** verbal, somatic

---

## Description
You touch a living creature that has 0 hit points.
The creature becomes stable.
This spell has no effect on undead or constructs.

**Sources:** PHB, page 277
